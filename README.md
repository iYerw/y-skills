# Y-Skills

我的私有技能仓库，用于 OpenClaw Agent。

## 目录结构

```
skills/
├── README.md
├── .gitignore
└── <skill-name>/
    ├── SKILL.md          # 技能定义（必需）
    ├── scripts/          # 脚本文件（可选）
    └── references/       # 参考文档（可选）
```

## 使用方式

Clone 到 OpenClaw workspace 的 skills 目录：

```bash
cd ~/.openclaw/workspace
git clone https://github.com/iYerw/y-skills.git skills
```

更新技能：

```bash
cd ~/.openclaw/workspace/skills && git pull
```
